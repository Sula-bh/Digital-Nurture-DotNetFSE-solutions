1. Explain the different types of linked lists (Singly Linked List, Doubly Linked List).

Answer:

The Different Types of Linked Lists are:

Singly Linked List: Each node contains data and a reference to the next node. Traversal is possible only in the forward direction.
Doubly Linked List: Each node contains data, a reference to the previous node, and a reference to the next node. Traversal is possible in both directions.

2. Analyze the time complexity of each operation.

Answer:

The Time Complexity Analysis is:

Operation		Time Complexity
Add (at end)	O(n)
Search			O(n)
Traverse		O(n)
Delete			O(n)

Where:

n = Number of tasks in the linked list.

3. Discuss the advantages of linked lists over arrays for dynamic data.

Answer:

The Advantages of Linked Lists are:

Dynamic size; no need to define capacity in advance.
Efficient insertion and deletion operations without shifting elements.
Better memory utilization for unpredictable data sizes.
Nodes can be allocated as needed during runtime.

Linked Lists are preferred when the number of elements changes frequently, whereas arrays are better for fast index-based access.
